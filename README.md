# 📸 LexPix Photography Portfolio ✨

A modern, responsive photography portfolio website showcasing professional photography services and gallery management. 🎨

## ✨ Features

- 📱 **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- 🔐 **Admin Dashboard**: Secure admin panel for managing:
  - 🖼️ Photo Gallery
  - 📅 Events
  - ⭐ Reviews
  - 📨 Contact Messages
- 🎯 **Dynamic Gallery**: Interactive photo gallery with category filtering
- 📆 **Event Management**: Upcoming events showcase and booking system
- 💬 **Review System**: Client testimonials and rating system
- 📬 **Contact Form**: Direct client communication channel

## 🛠️ Tech Stack

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript
- 💾 Local Storage for data persistence
- 🎯 FontAwesome for icons
